# Test Generation

Every action performed with `playwright-cli` generates corresponding Playwright TypeScript code in the output.

## Workflow
1. `playwright-cli open <url>`
2. `playwright-cli snapshot` to see references (`e1`, `e2`, etc.)
3. Perform actions: `playwright-cli fill e1 "user"`, `playwright-cli click e3`.
4. Copy generated `page.getByRole(...)` code from output into your `.spec.ts` file.

## Best Practices
- Use semantic locators (generated by default).
- Add assertions manually in the test file (e.g., `await expect(page).toHaveURL(...)`).
